# Database Creator

A Python utility for creating and managing SQLite databases.

## Features

- Create SQLite databases with custom table structures
- Predefined templates for common database scenarios
- Simple SQL wrapper functions for common operations

## Usage

Run the script and follow the prompts:

```
python database_creator.py
```

### Available Options

1. Create a new database with custom tables
2. Add data to existing tables
3. Query data from tables

## Database Templates

The script includes predefined templates for:

- Web store (customers, products, orders)
- Music library (artists, albums, songs)

## Requirements

- Python 3.6+
- SQLite3 (included in Python standard library)
